# MONTHLY-BUDGET-PLANNER
## Overview

The Budget Planner is a command-line tool designed to help users manage their monthly income and expenses. It allows you to track financial transactions, view summaries, and calculate your current balance, all while storing data persistently in a JSON file.

## Features

Add income and expenses with descriptions and amounts.

View detailed summaries of income, expenses, and the current balance.

Store transaction data in a JSON file to retain information across sessions.

Simple and easy-to-use command-line interface.

## Technologies/Tools Used

Python 3.x

JSON for data storage

Command-line interface (CLI)

## Steps to Install & Run the Project

Clone or download the project repository to your local machine.

Navigate to the project directory in the terminal.

Install any necessary dependencies (if applicable):
pip install -r requirements.txt
python budget_planner.py

## Instructions for Testing

Add sample income and expense transactions to see how the summary updates.

Use the "View Summary" option to check the calculated totals for income, expenses, and balance.

View the transaction history to ensure all transactions are recorded and displayed properly.
