## Problem Statement

Many individuals struggle to manage their finances effectively, often losing track of their income, expenses, and overall budget. This results in poor financial decision-making and financial stress. The Budget Planner addresses this problem by offering an easy-to-use, command-line tool that helps users track their monthly income and expenses, providing an overview of their financial status and helping them make more informed financial decisions.

## Scope of the Project

The Budget Planner is a personal finance management tool that allows users to:

Track their income and expenses.

View summaries of their financial situation, including total income, expenses, and balance.

Store transaction data persistently in a JSON file for future access.

View a history of transactions.

The project scope includes the following functionalities:

Adding income and expense transactions.

Viewing a summary of the current financial status.

Displaying a detailed transaction history.

Data persistence in a JSON file.

## Target Users

The Budget Planner is designed for:

Individuals who want to manage their personal finances and track their monthly budget.

Students, employees, freelancers, and anyone looking for a simple and efficient way to monitor their income and expenses.

Users with minimal technical expertise, as it is a command-line tool.

## High-Level Features

Add Income/Expense: Users can input income or expense transactions with descriptions and amounts.

View Financial Summary: The application calculates and displays the total income, total expenses, and the current balance.

View Transaction History: Users can see a detailed list of all their transactions, including date, type (income/expense), description, and amount.

Persistent Data Storage: All transactions are stored in a JSON file, ensuring data is preserved between sessions.

Simple Command-Line Interface: A straightforward and easy-to-use CLI that allows users to interact with the application and manage their budget.
